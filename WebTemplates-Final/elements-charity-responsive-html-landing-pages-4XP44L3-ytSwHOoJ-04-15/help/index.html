HTML Files and Structure

There are four files which includes All Elements file.

Html Structure

  1. This is a responsive layout
  2. <DIV>tag is used as container
  3. Each section is commented at just top and bottom of it
  4. Comments in HTML file exactly match with comments in CSS file
  5. Call-off scripts and styles are placed just top of the page
  6. Use recent version of any browser for better result.
  7. Tested in IE 10 & 11, Firefox, Chrome , Safari and Opera

To change the Fav Icon,Pick any variation of html template and open with any editor. Top of the page, short cut icon link has been given for fav icon. Here you change the fav icon name and place corresponding icon in the "images" folder. As well as, do the update of web site title and meta description for SEO point of view.
